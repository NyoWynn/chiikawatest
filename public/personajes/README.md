# Imágenes de personajes

Las imágenes están en **`resource/`** (formato `.webp`). La app las usa así:

- `resource/chiikawa.webp`
- `resource/hachiware.webp`
- `resource/usagi.webp`
- `resource/momonga.webp`
- `resource/rakko.webp`
- `resource/kurimanju.webp`
- `resource/shisa.webp`
- `resource/Furuhonya.webp` (F mayúscula)

Si falta la imagen de un personaje (pochette, roudou, ramen), se muestra el emoji por defecto.

**Dónde conseguir imágenes (respetando derechos):**
- Contenido propio o con permiso
- [Giphy](https://giphy.com) – busca "Chiikawa" (tier gratuito, revisa términos)
- [Tenor](https://tenor.com) – API gratuita de Google para GIFs
